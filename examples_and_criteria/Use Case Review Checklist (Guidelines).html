Use Case Model review checklist

 

Completeness

Are all use case definition fields filled in?

Are there any missing actors?

Are there missing use cases?

 

 

Correctness

Are the use case names expressed as active verbs (one word)?

Have nouns been used for all actor names?

Are all words used meaningful?

Is the diagram technically correct?

Are all functional requirements correctly captured?

 

 

Consistency

Are all use cases within the scope of the system?

Are all actors required for the system?
